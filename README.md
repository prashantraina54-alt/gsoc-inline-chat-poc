

# gsoc-inline-chat-poc
Proof of Concept for an Inline Contextual AI Chat feature using Node.js and the Gemini API.

## How to Run Locally
1. Clone the repository and run `npm install`
2. Create a `.env` file and add your Gemini API key: `API_KEY=your_key_here`
3. Run `node server.js` and open `http://localhost:8000`
